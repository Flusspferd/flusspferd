Flusspferd

Javascript bindings for C++

Flusspferd: Javascript bindings for C++, a Javascript interpreter and shell, and a Javascript/C++ module system. Also, Flusspferd includes a few Javascript classes and modules: binary blobs, XML, cURL, SQLite 3 and others. It is designed to support easy embedding of Javascript in C++ - and vice versa. It supports Spidermonkey as its backend, and possibly support for other engines will be added later.

Flusspferd is licensed under the MIT license (open source).

Our first release 0.5 is planned for April 30, 2009.

We have started on trying to support Windows, but we would appreciate help in getting there.

Please note that Flusspferd is subject to constant and major changes, so... don't be surprised if things change! We will be more focused on backwards compatibility once we have released.

In case you wonder, "Flusspferd" is a German word which translates to hippo. The reason we chose this word is that a book on Javascript shows a hippo on its cover, and Javascript has this animal naming tradition: Spidermonkey, Rhino, Squirrelfish and now Flusspferd.

Code

You can either fetch the code from our git repository

$ git clone git://github.com/ruediger/flusspferd.git

or download a snapshot from github.

Read the build instructions on how to build Flusspferd and its requirements. Currently Flusspferd only works on POSIX-type systems, being tested on Linux and OS X.
