Flusspferd

Javascript bindings for C++, a Javascript interpreter and shell, and a Javascript/C++ module system.

Flusspferd supports Spidermonkey as its backend, and possibly support for v8 or other engines will be added later.

In case you wonder, "Flusspferd" is a German word which translates to hippo. The reason we chose this word is that a book on Javascript shows a hippo on its cover, and Javascript has this animal naming tradition: Spidermonkey (two animals, even!), Rhino, and now Flusspferd.
